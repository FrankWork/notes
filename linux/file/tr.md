    [:alnum:] 所有的字母和数字

    [:alpha:] 所有的字母

    [:blank:] 所有的水平空格

    [:cntrl:] 所有的控制字符（非打印）

    [:digit:] 所有的数字

    [:graph:] 所有的可打印字符，不包含空格

    [:lower:] 所有的小写字母

    [:print:] 所有的可打印字符，包含空格

    [:punct:] 所有的标点符号

    [:space:] 所有的水平或垂直空格

    [:upper:] 所有的大写字母

    [:xdigit:] 所有的十六进制字符



    tr -d 删除字符

    tr -s 压缩字符（将连续重复的字符用一个字符表示）

    \b 空格

    \f 换页符

    \v 垂直制表符

    \NNN 八进制字符 NNN
